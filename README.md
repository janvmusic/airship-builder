# airship-builder

### Objective
- Understand how builder, factory & abstract factory works creating a project that will create airships and their parts

### Tech Stack
- Maven
- Java 8
- JSON

### Patterns
- Builder
- Factory
- Abstract Factory

### Sorting
- By Rank
- By Price
- By Weight
- By Fuel
